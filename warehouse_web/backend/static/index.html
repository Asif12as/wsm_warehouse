Warehouse Management API

Welcome to the Warehouse Management API. This API provides endpoints for managing warehouse data, including SKU mapping and sales data processing.

Available Endpoints

GET /api/health

Health check endpoint to verify the API is running.

POST /api/upload/mapping

Upload a SKU mapping file.

POST /api/upload/sales

Upload a sales data file.

POST /api/process

Process the uploaded mapping and sales files.

GET /api/results

Get processing results.

GET /api/export

Export processed data in CSV or XLSX format.

Frontend Application

The frontend application is available at: http://localhost:3000
